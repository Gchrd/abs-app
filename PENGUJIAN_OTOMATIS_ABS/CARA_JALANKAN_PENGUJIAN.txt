--------------------------------------------------------------------------------
PETUNJUK CARA MENJALANKAN PENGUJIAN OTOMATIS:
--------------------------------------------------------------------------------

Langkah-langkah untuk mengeksekusi tes di komputer Anda:

1. Buka Terminal / Command Prompt / PowerShell.
2. Pindah ke direktori backend proyek ABS:
   cd "d:\Mayora\Laporan Magang\LateX laporan magang\abs-source-full\backend"

3. Pastikan dependensi pengujian (pytest dan library backend) sudah terinstal:
   pip install pytest fastapi sqlalchemy pyjwt

4. Jalankan perintah pytest dengan verbose mode (-v) untuk melihat seluruh 47 kasus uji:
   pytest tests/ -v

5. Hasil eksekusi akan muncul di layar. Anda akan melihat 47 indikator hijau "PASSED" yang menandakan seluruh sistem backend dan fitur Anda lulus pengujian dengan sukses 100%.
